# ReserBayan Monorepo

This is the monorepo for ReserBayan - Online Barangay Document Request System.

## Structure

- `/frontend` - Next.js application
- `/backend` - Spring Boot API

## Running the Project

First, install root dependencies:

```bash
npm install
```

Then run:

### Start everything

```bash
npm run dev
```

### Start frontend only

```bash
npm run dev:frontend
```

### Start backend only

```bash
npm run dev:backend
```
